Hide unwanted UI elements in Pear with custom CSS

Note: These selectors may become outdated after YouTube updates. Test the CSS after any major YouTube changes.
How to use?
  1. Select which elements to hide
  2. Copy or download the CSS
  3. In Pear: Options → Visual Tweaks → Theme
  4. Import the CSS file or add to existing theme
  5. Navigation → Restart App to apply changes
